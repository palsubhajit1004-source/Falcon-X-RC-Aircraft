## About the Project
Falcon X is a RC aircraft designed and developed by me as a personal project.  
The goal of this project is to gain hands-on experience in aircraft design, aerodynamics, electronics, and flight testing.

## Specifications
- **Wingspan:** 80 centimetres 
- **Motor:** 1400 KV Brushless  
- **Battery:** 11.1V 3S LiPo (1000 mAh)  
- **Control:** Ailerons, Elevator, Rudder  
- **Landing Gear:** Tricycle (Nose wheel)  
- **Transmitter:** Radiolink T8FB  

## Electronics Used
- 1400KV Brushless DC Motor  
- 30A ESC  
- 11.1V 1000mAh 3s LiPo Battery  
- 9g Servos (Aileron, Elevator, Rudder)  
- T8ES Receiver
